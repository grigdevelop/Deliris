# Competitor Feature Analysis

This document outlines the key features of major competitors in the ride-hailing and delivery space.

## Uber

### Rider Features
- **Ride Booking**: Book rides for now or schedule for later.
- **Real-Time Driver Tracking**: See the driver's location in real-time on a map.
- **Fare Splitting**: Split the cost of a ride with other passengers.
- **Multiple Drop-off Locations**: Add multiple stops in a single trip.
- **Multiple Payment Options**: Pay with cash, credit card, or mobile wallet.
- **Preferred Driver**: Option to select a preferred driver.
- **Driver Rating and Review**: Rate and review drivers after a trip.
- **Trip History**: View past trips and receipts.
- **Saved Destinations**: Save frequently used addresses.
- **Safety Features**: SOS button, live ride sharing, trip verification.

### Driver Features
- **Flexible Schedule**: Drive whenever they want.
- **Earnings Tracker**: See earnings after every trip.
- **In-App Navigation**: Turn-by-turn directions.

## Glovo

### Core Features
- **Multi-Category Ordering**: Order from restaurants, supermarkets, pharmacies, and more.
- **Real-Time Order Tracking**: Track orders on a map.
- **Anything Delivery**: Pick-up and drop-off service for any item.
- **Scheduled Deliveries**: Schedule orders in advance.
- **Smart Recommendations**: AI-based suggestions for orders.

### User Features
- **Saved Addresses & Payment Methods**: One-tap reordering.
- **In-App Chat**: Communicate with the delivery partner.
- **Promo Codes & Loyalty Programs**: Discounts and referral bonuses.

### Partner Features
- **Vendor Dashboard**: Manage orders, revenue, and inventory.
- **In-App Notifications**: Instant alerts for new orders.
- **Menu/Inventory Management**: Easily update offerings.

### Courier Features
- **Task Management Panel**: Clear instructions for each delivery.
- **Route Optimization**: Integrated navigation to find the fastest routes.
- **Earnings Tracker**: Transparent view of earnings, bonuses, and tips.

## Yandex Go (Taxi & Eats)

### Yandex Taxi
- **Ride Hailing**: Request rides through the app.
- **Multiple Service Classes**: Choose from different car types and service levels.
- **Upfront Pricing**: See the ride price before booking.
- **Multiple Payment Options**: Pay with cash or card.
- **Add Stops**: Add multiple stops to a trip.
- **Request Multiple Rides**: Order several cars at once.
- **Public Transport Schedule**: View bus schedules in the app.

### Yandex Eats
- **Food and Grocery Delivery**: Order from restaurants and stores.
- **Fast Delivery**: Average delivery time of 32 minutes.
- **Real-Time Order Tracking**: Track the courier on the map.
- **User-Friendly Interface**: Easy to browse menus and place orders.
- **Search and Filters**: Find specific cuisines, dishes, or dietary options.
- **Restaurant Profiles and Ratings**: Make informed choices.
